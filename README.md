# Walking Calculator

**一个辅助你在朋友之间记账的 App 。记录好每笔开销的明细，并帮你计算出最终的债务去向。已适配深色模式。**

## 🌟 Build & Run

在 **/src/config.js** 中修改 `host` 地址

```js
global.host = 'http://192.168.50.44:3001'		// 前端 host
global.srvHost = 'http://192.168.50.44:3000'		// 后端 host
```
